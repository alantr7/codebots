# Overview
CodeBots is a Minecraft plugin that allows you to create and program bots inspired by ComputerCraft's turtles. It uses a custom-made scripting language with specifications below.

## Features
Bots can place and break blocks, check what's around them, move, chat, display text on monitors, connect to Torus and
control machines. There will be a proper API in the future to let developers create custom addons.
